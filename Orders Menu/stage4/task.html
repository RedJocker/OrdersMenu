Description

We don't want to just list menu items on our app. We should also give waiters a way to adjust the quantity of a certain menu item that will be ordered. Also we should limit the amount ordered considering the amount in stock otherwise the kitchen won't be able to complete the order.

Let's improve the MenuItem component so that waiters can click to increase/decrease amount ordered and see the what is the current amount. Change text to red if the amount ordered is on max limit.

Objectives

  1. Add a Text with text "+" on same line as the existing Text with text "Fettuccine"
  2. Add a onClick to "+" so that the amountOrdered increases after it is clicked
  3. Add a Text on same line as the existing Text with text "Fettuccine" set text to the current amountOrdered
  4. The initial amount ordered is 0
  5. Add a Text with text "-" on same line as existing Text with text "Fettuccine"
  6. Add a onClick to "-" so that the amountOrdered increases after it is clicked
  7. If the amountOrdered has not yet achieved the amountStock make "Fettuccine" text color black
  8. If the amountOrdered is equal to the amountStock make "Fettuccine" text color red
  9. amountOrdered cannot be negative nor bigger than amountStock
  10. consider the amountStock of "Fettuccine" to be 5
