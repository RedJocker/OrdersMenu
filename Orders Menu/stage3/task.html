Description

We don't want to just list menu items on our app. Give waiters a way to adjust the amount ordered of a certain menu item. Also limit the amount ordered considering the amount in stock otherwise the kitchen won't be able to complete the order.

Let's improve the MenuItem composable so that waiters can click to increase/decrease amount ordered and see the what is the current amount. Change text to red if the amount ordered is on max limit.

Objectives

  1. Consider amountOrdered to be the quantity to be ordered of a certain menu item. The initial amountOrdered value is 0;
  2. Consider amountStock to be the quantity in stock of a certain menu item. The initial amountStock for "Fettuccine" is 5;
  3. Add a Text with text "+" on same line as the existing Text with text "Fettuccine";
  4. Add a onClick to "+" so that the amountOrdered increases after it is clicked;
  5. Add a Text on same line as the existing Text with text "Fettuccine" set text to the current amountOrdered;
  6. Add a Text with text "-" on same line as existing Text with text "Fettuccine";
  7. Add a onClick to "-" so that the amountOrdered decreases after it is clicked;
  8. If the amountOrdered has not yet achieved the amountStock make "Fettuccine" text color black
  9. If the amountOrdered is equal to the amountStock make "Fettuccine" text color red
  10. amountOrdered of a certain menu item cannot be negative nor bigger than amountStock

obs: for simplification reasons we are not considering the subject of dark-mode on our requirements.
